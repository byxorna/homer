Usage

Send a DNS query as a base64url-encoded UDP wire-format message:

GET /.well-known/dns-query?ct=application/dns-udpwireformat&body=BASE64URL_DNS_MESSAGE
Content-Type: application/dns-udpwireformat
Accept: application/dns-udpwireformat

Client Setup

Container (recommended)

The container image ships both homer-client and homer-server. Pull it and run the client directly:

podman pull docker.io/byxorna/homer-server:latest

Create a client config pointing at this server:

# config.yaml
listenaddr: 0.0.0.0
listenport: 53
doh:
  url: https://YOUR_DOMAIN

Run the client, binding port 53 on the host:

podman run -d --name homer-client \
  --restart unless-stopped \
  -p 127.0.0.1:53:53/udp \
  -v $(pwd)/config.yaml:/config/client.yaml:ro \
  --entrypoint homer-client \
  docker.io/byxorna/homer-server:latest \
  -config /config/client.yaml

Then set nameserver 127.0.0.1 in /etc/resolv.conf and query normally with dig.

To stop: podman stop homer-client

From source

See the project README for build instructions.

# config.yaml
listenaddr: 127.0.0.1
listenport: 53
doh:
  url: https://YOUR_DOMAIN
sudo homer-client -config config.yaml

Then set nameserver 127.0.0.1 in /etc/resolv.conf and query normally with dig.
